Aplicativo mobile desenvolvido em React Native com consumo de API externa, navegação entre telas e integração com Firebase para armazenamento de favoritos.

Funcionalidades: navegação entre telas com React Navigation, consumo da PokeAPI para listagem de dados, sistema de favoritos com Firebase, CRUD completo (criar, listar, atualizar e remover favoritos) e animações simples na interface.

API utilizada: PokeAPI

Firebase - Utilizado para armazenar e gerenciar os itens favoritos com operações de CRUD em tempo real.

Navegação - O app possui 3 telas: Home, Listagem e Favoritos.

Tecnologias utilizadas: React Native, React Navigation, Firebase Firestore, JavaScript, PokeAPI e Animated API.
